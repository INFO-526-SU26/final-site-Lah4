# Data

- **\[https://github.com/williamorim/pokemon\]**: This package provides a dataset of Pokemon information in both English and Brazilian Portuguese. The dataset contains 949 rows and 22 columns, including information such as the Pokemon’s name, ID, height, weight, stats, type, and more.

# Codebook for Pokemon Dataset

## Variable Names and Descriptions:

- **id**: The unique ID of each Pokémon.
- **pokemon**: The name of each Pokémon.
- **species_id**: The species ID of each Pokémon.
- **height**: The height of each Pokémon.
- **weight**: The weight of each Pokémon.
- **base_experience**: The base experience of each Pokémon.
- **type_1**: The primary type.
- **type_2**: The secondary type.
- **hp**: The HP (hit points).
- **attack**: The attack points.
- **defense**: The defense points.
- **special_attack**: The special attack points.
- **special_defense**: The special defense points.
- **speed**: The speed.
- **color_1**: The primary color of each Pokémon.
- **color_2**: The secondary color of each Pokémon.
- **color_f**: The final color of each Pokémon.
- **egg_group_1**: The primary egg group.
- **egg_group_2**: The secondary egg group.
- **url_icon**: The URL for the icon of each Pokémon (if available; missing the leading `https:`).
- **generation_id**: The generation ID of each Pokémon.
- **url_image**: The URL for the image of each Pokémon.
